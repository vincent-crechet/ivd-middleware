# System Admin Portal

Portal for platform operators (not laboratory users).

## Features

- Create new laboratory tenants
- View all tenants
- Monitor system health across tenants
- Deactivate tenants
- Generate tenant API keys

## Tech Stack

- Vue.js 3
- Vue Router
- Pinia
- Axios

## Running

```bash
npm install
npm run dev
```

## Note

This is optional for MVP. Can start with API-only access for system administrators.
